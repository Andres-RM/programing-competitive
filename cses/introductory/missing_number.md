# Missing Number

**Problema:** Te dan todos los números del 1 al n excepto uno. Tu tarea es encontrar el número faltante.

**Input:**
- Primera línea: un entero n
- Segunda línea: n-1 números

**Output:**
- El número faltante

**Ejemplo:**
```
Input:
5
2 3 1 5

Output:
4
```

**Límites:**
- 2 ≤ n ≤ 2·10^5

**Link:** https://cses.fi/problemset/task/1083
